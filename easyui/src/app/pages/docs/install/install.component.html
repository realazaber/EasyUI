Install

Easy UI is based on top of tailwind. To install it you will need to.

First install tailwind css into your project

Make sure the tailwind config file is at the same level as your src folder.

Setup

Set up your tailwind config with your theme colors.

You can find official tailwind guide here

You will then need to add Easy UI to your project.

Then import the EasyUIModule to your components where itll be used.

Now whenever you start a < you will most likely (depending if you have intellisense) see the wide array of components to choose from!
